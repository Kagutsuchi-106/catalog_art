# Каталог современного искусства — Тирасполь

Веб-приложение для каталогизации и поиска событий современного искусства в Тирасполе.

##  Описание проекта

Каталог современного искусства — это интерактивная платформа, которая позволяет пользователям:
- Просматривать актуальные события (выставки, лекции, мастер-классы, перформансы, кинопоказы)
- Фильтровать события по типу, жанру, дате, цене, району
- Искать события по названию, ключевым словам и именам художников
- Просматривать профили художников с портфолио
- Сохранять избранные мероприятия
- Рассчитывать бюджет посещения мероприятий
- Просматривать карту культурных пространств


### Фильтрация и поиск
- **Фильтры по типу**: Выставки, Лекции, Мастер-классы, Перформансы, Кинопоказы
- **Фильтры по жанру**: Современная живопись, Фотография, Скульптура и инсталляция, Видео-арт и медиа-искусство, Графика и иллюстрация
- **Фильтры по датам**: Сегодня, На этой неделе, В этом месяце, Выбор диапазона дат
- **Фильтры по цене**: Бесплатные, До 300₽, 300-700₽, Более 700₽
- **Фильтры по районам**: Центральный, Левобережный, Северный
- **Поиск**: По названию, ключевым словам, имени художника
- **Сортировка**: По дате, цене, алфавиту, популярности

### Персонализация
- Сохранение избранных мероприятий в localStorage
- Сохранение настроек фильтров
- Калькулятор бюджета с учетом категории билета

### Доступность
- Полная поддержка ARIA-атрибутов
- Семантическая HTML5 разметка
- Клавиатурная навигация
- Поддержка скринридеров

##  Технологии

- **HTML5** с семантической разметкой
- **CSS3** с использованием CSS Grid, Flexbox, CSS-переменных
- **JavaScript (ES6+)** без фреймворков
- **localStorage** для сохранения данных пользователя
- **Schema.org** разметка для SEO

##  Запуск проекта

1. Клонируйте репозиторий:
```bash
git git@github.com:Kagutsuchi-106/catalog.git
cd project_news
```

2. Откройте `index.html` в браузере 

##  Данные

Проект использует JSON для хранения данных:
- **20-25 мероприятий** с полной информацией (тип, жанр, даты, цены, кураторы, концепции)
- **6 художников** с биографиями, концепциями и портфолио
- **8 культурных пространств** с координатами и районами

Все данные хранятся в `scripts/data.js` и могут быть легко расширены.
